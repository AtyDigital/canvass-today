# Canvass Today

Political Campaign Intelligence Platform — Interactive Prototype

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Import the repo in [Vercel](https://vercel.com/new)
3. Deploy with default settings — no build step needed

Or use the Vercel CLI:
```bash
npx vercel --prod
```

## Structure

- `index.html` — Single-page application with hash-based routing
- `style.css` — Design system tokens and component styles
- `base.css` — Reset and base styles
- `vercel.json` — Vercel deployment configuration
